LAS HERRAMIENTAS ETL
ETL es el proceso por el cual se extraen datos o información para su transformación, ya sea combinarse con otros datos u origenes, realizar filtros o cualquier otro procedimiento que cambie su forma inicial mas no su valor de dato como tal, y posteriormente cargar la información, sea en el medio de origen u otro medio (migración).
Extracción
Es el copiado o extracción inicial del segmento de información o datos escogidos, los cuales deben cumplir con las condiciones requeridas de integridad o extensión, los cuales deberán ser tenidos en cuenta para la siguiente fase. Dependiendo del lugar de alamacenamiento de la información de origen, sea una base de datos en un repositorio, Data Warehaouse, o cualquier otro almacen de datos, la extracción se programa en momentos en los que el sistema como tal no tiene carga de transacciones o éstas son muy bajas, debido a la necesidad de realizar un copiado optimo.
Transformación
La transformación de la información copiada se puede dar aplicando filtros, cambios de tipos de datos o interpretaciones, formatos para cadenas de texto, fechas y distintos tipos de valores numéricos. Este tipo de procesos se llevan a cabo usando herramientas tecnológicas y de software como lo son Microsoft Power BI o Pentaho Data Integration Suite.
Carga
La carga de los datos se debe dar respetando el destino de los mismos, en el caso de migraciones de datos entre distintas bases de datos se debe realizar una conversión respetando cada lenguaje y sistema de referencia, es importante que haya un plan de backup y recuperación de logs de errores en el caso de presentarse inconsistencias en el destino.
