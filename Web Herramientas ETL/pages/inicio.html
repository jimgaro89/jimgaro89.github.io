Información Curso Herramientas ETL
El curso Herramientas ETL busca en principio, entregar herramientas para la aplicación de la extracción, transformación y carga de información principalmente orientado a poblamiento de un Data Warehouse.

El tema será abordado teniendo en cuenta los diferentes tipos de bases de datos que puedan ser usados en este proceso y la creación de las mismas como por ejemplo SQL SERVER. .

El curso de Herramientas ETL es vital para el estudiante que pretenda adelantar procesos de migración de datos, análisis de datos apoyandose en herramientas de BI o el uso de tecnologías OLAP, así como minería de datos y demás procesos que usen el dato para agregarle valor a través del proceso de transformación que puede darse usando herramientas ETL.
